# WriteRowConnections

Key attributes:
- `Item` — spreadsheet selector
- `Range` — sheet name or range
- `ArrayRow` — `object[]` variable containing row values
- `DataType` — `"ArrayRow"`
- `HasHeaders` — `"True"` / `"False"`
- `WriteMode` — `"Append"` or `"Overwrite"`
- `RowPosition` — target row index
