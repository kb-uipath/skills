# Orchestrator Presentation Rules

- **Jobs** — display as process name + version. Show job key in parentheses only when needed for commands.
- **Folders** — display name, not folder ID or folder key
- **Queues** — display name, not queue ID
- **Machines** — display name, not machine ID
- **Robots** — display name, not robot ID
- **Assets** — display name, not asset ID
- **Settings** — use the UI-facing label (e.g., "Job execution timeout" not `MaxExpectedRunningTimeSeconds`). If no UI label found, describe the setting functionally.
