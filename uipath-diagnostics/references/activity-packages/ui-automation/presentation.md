# UI Automation Presentation Rules

- **Activities** — use the display name (e.g., "Click", "Type Into", "Get Text"), not the fully qualified class name (e.g., `UiPath.UIAutomationNext.Activities.NClick`)
- **Selectors** — show the human-readable target description (e.g., "Click 'Submit' button") alongside the selector XML when relevant. Do not show raw selector XML as the primary identifier.
- **Elements** — describe by what the user would see (e.g., "the 'Collaboration' link"), not by selector attributes alone
